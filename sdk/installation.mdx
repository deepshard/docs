## Prerequisites

- Python 3.12 or higher
- A Truffle¹ device
- The [Symphony desktop client](/client/overview), with your device already onboarded

## Install Python

<Tabs>
  <Tab title="macOS">
    ```bash
    brew install python@3.12
    ```

    Verify:
    ```bash
    python3 --version
    ```
  </Tab>
  <Tab title="Windows">
    Download and install Python from [python.org](https://www.python.org/downloads/), or use Chocolatey:

    ```powershell
    choco install python --version=3.12.0
    ```

    Verify:
    ```powershell
    python --version
    ```
  </Tab>
  <Tab title="Linux">
    ```bash
    sudo apt update
    sudo apt install python3.12 python3.12-venv python3-pip
    ```

    Verify:
    ```bash
    python3 --version
    ```
  </Tab>
</Tabs>

## Set up a virtual environment

<Tabs>
  <Tab title="venv">
    ```bash
    python3 -m venv .venv
    ```

    <Tabs>
      <Tab title="macOS / Linux">
        ```bash
        source .venv/bin/activate
        ```
      </Tab>
      <Tab title="Windows">
        ```powershell
        .venv\Scripts\Activate.ps1
        ```
      </Tab>
    </Tabs>
  </Tab>
  <Tab title="uv">
    If you prefer [uv](https://github.com/astral-sh/uv) for faster package management:

    ```bash
    # install uv if you don't have it
    curl -LsSf https://astral.sh/uv/install.sh | sh

    # create and activate a venv
    uv venv
    source .venv/bin/activate   # macOS / Linux
    # or
    .venv\Scripts\Activate.ps1  # Windows
    ```
  </Tab>
</Tabs>

## Install Truffile

With your environment activated:

```bash
pip install truffile
```

Verify the installation:

```bash
truffile help
```

You should see the Truffile welcome panel listing every command (`scan`, `connect`, `deploy`, `chat`, `infer`, and the rest). If you see the panel, you're ready to go.

<Note>
Make sure you've onboarded your Truffle with the Symphony desktop client and grabbed your User ID from **Settings → About** before running `truffile connect`.
</Note>

## Building apps with Claude Code

If you're cloning the [truffile repo](https://github.com/deepshard/truffile) to work on apps, there's a bundled skill called **`truffle-app-creator`** (`skills/SKILL.md`) that a Claude Code agent can use to build a Truffle app end-to-end from a description. Give it a prompt like *"Create a Truffle app that connects to \<service\>"* and it'll handle scaffolding, implementing the tools, wiring up auth, and running `truffile validate` + `truffile deploy`.
